# 🚗 Car Price Prediction Web App

A Machine Learning-powered Flask web application that predicts the selling price of used cars based on various input features such as manufacturing year, fuel type, transmission type, seller type, ownership history, and kilometers driven.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/roshankodi/car-price-prediction.git
```

### 2️⃣ Navigate to Project Directory

```bash
cd car-price-prediction/car_price_prediction
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Flask App

```bash
python main.py
```

### 5️⃣ Open in Browser

```bash
http://127.0.0.1:5000
```
